# DocuAId
## Your personal guide to understanding German bureaucracies and their documents

This Repository is the working directory for the Impact lab hackathon on 23.07.2026

People moving to Germany often struggle with understanding what the bureaucracy demands from them to start working and living in Germany. We built a Web App that supports people arriving in Germany by translating and explaining Documents presented to them by the German Immigration and Work Agencies, as well any other official documents.
The Tool uses the Claude API to analyze the documents as well as generating next steps based on the Provided Dataset.
The tool is meant to assist people who are none German speakers in simplifying the complicated web of bureaucracy and documents into manageable and actionable next steps.
